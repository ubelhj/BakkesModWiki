Find the ID of the current playlist with 
```cpp
ServerWrapper sw = gameWrapper->GetOnlineGame();
if (!sw) return;
GameSettingPlaylistWrapper playlist = sw.GetPlaylist();
if (!playlist) return;
int playlistID = playlist.GetPlaylistId();
```
